# 📖 About the Project

Schools today use multiple applications for communication, attendance, academics, meetings and reports. Because these systems are disconnected, teachers, students and parents often struggle with communication and accessing important information.

**VidyaSetu AI** solves this problem by bringing everything together into one intelligent platform.

It is an AI-powered School Management System where students, teachers, parents and administrators can communicate in real time while also getting AI-powered insights and automation.

Instead of simply storing data, VidyaSetu AI helps schools make better decisions.

---

# 🎯 Problem Statement

Most schools still face problems like:

- Poor communication between teachers and parents
- Student performance is tracked manually

# ⚙ Environment Variables

Create a `.env` file in the project root.

```env
GOOGLE_API_KEY=YOUR_GEMINI_API_KEY

MONGODB_URI=mongodb+srv://USERNAME:PASSWORD@cluster.mongodb.net/VidyaSetuAI

JWT_SECRET=your_secret_key

PORT=3000

FASTAPI_URL=http://localhost:8000
```

# 🚀 Installation Guide

## 1. Clone Repository

```bash
git clone <repository-url>

cd VidyaSetu-AI
```

---

## 2. Install Node Packages

```bash
npm install
```

---

## 3. Setup Python Environment

```bash
cd ai

python -m venv venv
```

---

### Windows

```bash
venv\Scripts\activate
```

---

### Linux / Mac

```bash
source venv/bin/activate
```

---

## 4. Install Python Packages

```bash
pip install -r requirements.txt
```

---

## 5. Configure MongoDB Atlas

- Create MongoDB Atlas Cluster

- Create Database User

- Add Current IP

- Copy Connection String

- Paste it inside

```
.env
```

---

## 6. Start AI Server

```bash
cd ai

uvicorn app:app --reload
```

Runs on

```
http://localhost:8000
```

---

## 7. Start Frontend + Backend

Go back to root directory

```bash
npm run dev
```

Runs on

```
http://localhost:3000
```
